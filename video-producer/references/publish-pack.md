# Publish Pack

Create `exports/publish_pack.md` after QC.

## Required Sections

1. Final export files and durations.
2. Title options by platform.
3. Cover text options.
4. Description / intro copy.
5. Hashtags / keywords.
6. Pinned comment.
7. Chapter list if long-form.
8. Cutdown plan: 15s, 30s, 60s variations.
9. Rights/attribution notes.
10. Experiment notes: hook, title, cover hypothesis.

## Platform Hints

- Douyin / TikTok: strong first line, visual hook, short cover text, fast captions.
- Bilibili: clearer title, chapter context, source/attribution notes, longer description.
- YouTube Shorts: keyword-rich title, compact description, clean captions.
- YouTube long-form: chapters, thumbnail rationale, source notes, pinned comment.
