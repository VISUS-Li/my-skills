# Storyboard Richness Rubric

Use this before rendering and when revising visual quality.

## Scoring dimensions

Score each major segment 0-3:

1. **Narrative role**: does the segment change what the viewer knows or feels?
2. **Visual metaphor**: is there a concrete visual idea beyond text?
3. **Shot progression**: does it include establish/detail/reveal or equivalent?
4. **Frame composition**: does it define focal zone, negative space, and ratio allocation?
5. **Depth**: foreground/midground/background/parallax or equivalent layering.
6. **Asset specificity**: planned icons/images/screenshots/charts/B-roll/textures/SFX.
7. **Motion motivation**: movements correspond to meaning, not random animation.
8. **Edit reason**: transition/cut is motivated by information, emotion, proof, contrast, or rhythm.
9. **Text discipline**: readable text is short and hierarchically clear.
10. **Style coherence**: palette, type, image style, and motion belong to one world.

## Pass thresholds

- 24-30: rich enough to render.
- 18-23: acceptable draft, revise before final.
- <18: likely to look empty/PPT-like; revise storyboard, shotlist, and assets first.

## Mandatory anti-empty fixes

If a segment scores low, apply at least two:

- Add a concrete visual object: product, chart, map, screenshot, icon system, metaphor prop.
- Split one card into three shots: context, detail, reveal.
- Add 3-layer depth: background texture, main object, foreground annotation.
- Replace bullet list with a spatial layout: matrix, pipeline, timeline, stack, map, orbit, split-screen.
- Add a camera move with meaning: push-in for importance, pull-out for system, parallax for exploration, whip for contrast.
- Add sound/VFX cues to support edit rhythm.

## Do not render final if

- No art direction file exists.
- Storyboard has text-only segments.
- Shotlist has fewer shots than segments.
- Asset manifest has no usable assets.
- Aesthetic score is below the project threshold.
